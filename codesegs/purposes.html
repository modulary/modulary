The following are the purposes for various code segments. Each code-segment is a PHP file called from within a function, located in this directory, who's filename is the prefix "cs-" followed by the name of the code-segment followed by ".php".

The return value of the script will be returned by the calling function as well.

The code-segments are called in require mode so that there will (if the server is so-configured) be error-output in the event of a missing file.

The $codeparam local-variable is where the identifier of the code-segment is stored.

act_as_logged_on tbi This code-segment will be called if and only if a user is presently logged on to the system.
but_not_logged_on tbi This code-segment will be called unless a user is presently logged on to the system.
is_logged_on ip This code-segment will return FALSE if no user is presently logged on to the system. However, if a user is presently logged on to the system, it will return TRUE after an object named 'userinf' is created in the anchor object with all relevant user information.
